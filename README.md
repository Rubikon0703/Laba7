##Задача 1
###Текст задачи
На вход программы подаются фамилии и имена учеников. Известно, что общее количество учеников не превосходит 100. В первой строке вводится количество учеников, принимавших участие в соревнованиях, N. Далее следуют N строк формата: <Фамилия> <Имя>. Требуется написать программу, которая формирует и печатает уникальный логин для каждого ученика по правилу: если фамилия встречается первый раз, то логин – это данная фамилия, если фамилия встречается второй раз, то логин – это фамилия, в конец которой приписывается число 2 и т.д.

###Алгоритм решения
Ввести целое число N – количество учеников.

Создать словарь surnameCount (ключ – фамилия, значение – количество повторений).

Создать список logins для хранения логинов в порядке ввода.

Для i от 1 до N:

Запросить строку вида "Фамилия Имя".

Разбить строку на части по пробелу. Взять первую часть – фамилию.

Если фамилия отсутствует в словаре:

Добавить её в словарь со значением 1.

Добавить в список логинов фамилию.

Иначе:

Увеличить значение для этой фамилии на 1.

Сформировать логин = фамилия + текущее значение счётчика.

Добавить логин в список.

Вывести все логины в порядке ввода.
